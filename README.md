# pokemon-search
A simple and beginner-friendly web app that lets you search for Pokémon by name or ID using the PokéAPI. It displays basic information like type, height, weight, and an image in a clean and easy-to-use interface built with plain HTML, CSS, and JavaScript.

## How to clone the repository

git clone https://github.com/tulio-ferreira21/pokemon-search
cd pokemon-search
